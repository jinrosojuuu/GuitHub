# 🎸 GuitHub

GuitHub is a responsive guitar store website developed using **HTML, CSS, Bootstrap, JavaScript, PHP, and MySQL**. This project was created as a final requirement for the Web Development course.

## Features

- Responsive multi-page website
- Guitar product showcase
- About Us and Contact pages
- Database-driven chatbot
- Admin Panel with CRUD operations
- Mobile and desktop friendly design

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- PHP
- MySQL
- XAMPP
 
## Installation

1. Install XAMPP.
2. Copy the project folder to `xampp/htdocs/`.
3. Start Apache and MySQL.
4. Create a database named `guithub_db`.
5. Import `database.sql`.
6. Open your browser and visit: http://localhost/GuitHub


## Admin Features (CRUD)

- Add chatbot responses
- View chatbot records
- Edit chatbot responses
- Delete chatbot responses

## Chatbot

The chatbot retrieves responses from a MySQL database and helps users with common inquiries about guitars, brands, store information, payments, and shipping.
